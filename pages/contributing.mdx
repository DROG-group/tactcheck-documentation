# Contribution Guide

We welcome contributions to TactCheck! Whether you want to report a bug, suggest a feature, or submit a pull request, your input is valuable.

## How to Contribute

1. **Fork the Repository**: Start by forking the relevant TactCheck repository (Gamepatch, Express backend, or documentation).

2. **Create a Branch**: Create a new branch for your feature or bug fix.
   ```bash
   git checkout -b feature-branch
   ```
