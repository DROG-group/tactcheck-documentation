# Frequently Asked Questions (FAQ)

## 1. What is TactCheck?

TactCheck is an open-source project developed by [DROG Group](https://drog.group) to simulate the spread of propaganda, hoaxes, and disinformation on social media platforms using Mastodon.

## 2. What is the purpose of Gamepatch?

Gamepatch is a Ruby on Rails system that injects simulated conversations into Mastodon. It’s a key component of TactCheck, enabling controlled experiments in social media dynamics.

## 3. Can I run TactCheck on any Mastodon instance?

TactCheck is designed to work with Mastodon (Glitch Edition). While it may work with other Mastodon versions, it’s optimized for the Glitch Edition.
