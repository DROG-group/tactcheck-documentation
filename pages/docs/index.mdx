## About TactCheck

TactCheck, created by [DROG Group](https://drog.group), simulates how social media can be used to spread propaganda, hoaxes, and disinformation. This project uses tools like Gamepatch and Mastodon to create a controlled environment for studying digital misinformation.

Explore the documentation to get the most out of TactCheck.
